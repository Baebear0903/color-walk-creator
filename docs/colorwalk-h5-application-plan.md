# Colorwalk H5 生成器实施计划

## 目标

构建一个可在 iPhone Safari 打开的纯前端 H5 工具。用户上传单张静态图片后，系统自动提取画面核心色，一键生成可长按保存的 Colorwalk 静态打卡图。

## 实施范围

- 使用静态前端实现：`index.html`、`styles.css`、ES modules。
- 自动取色策略：采样上传图片，按色值占比排序，并在候选色中尽量拉开色系；除非原图用色单调，避免五个候选色都来自相近色系。
- 编辑能力：主色、候选色卡、色板位置、内容文本、字体预设均可手动调整。
- 版式：横图生成 `1080x1440`，图片区保持 `3:2`，纯色色板默认在上；竖图生成 `1440x1080`，图片区保持 `2:3`，纯色色板默认在右。`4:3/3:4` 素材会居中裁切为 `3:2/2:3`。
- 导出：生成 PNG 静态图并显示为 `<img>`，用户在手机浏览器中长按图片保存。

## 界面方案

- 视觉方向采用多巴胺彩色：粉、黄、绿、蓝、紫作为界面能量色。
- 工作台保持移动端优先：上方预览，下方编辑，按钮尺寸满足单手点击；横图显示上/下色板位置，竖图显示左/右色板位置。
- 字体预设包括 Playfair、Cormorant、Baskerville、Typewriter、Songti、Kaiti，并提供系统字体回退。
- 内容文本使用支持换行的编辑框，默认内容为自动提取的 HEX 色值。

## 验收标准

- 可上传图片并自动生成候选主色。
- 上传非图片文件时给出明确提示。
- 默认内容文本带出主色 HEX，且可以手动换行修改。
- 切换字体、主色和文本后预览即时更新。
- 点击生成后得到静态图片，长按图片可保存。
