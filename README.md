# Colorwalk H5

手机浏览器用的 Colorwalk 打卡图生成器。上传一张静态图片后，页面会自动提取核心色，按素材横竖方向生成静态打卡图。生成结果会以图片形式展示，手机浏览器中长按即可保存。

## 线上地址

```text
https://baebear0903.github.io/color-walk-creator/
```

## 版式规则

- 横向素材输出 `1080x1440`，图片区域保持 `3:2`，色板默认在上，也可切到下。
- 纵向素材输出 `1440x1080`，图片区域保持 `2:3`，色板默认在右，也可切到左。
- `4:3` 或 `3:4` 素材会居中裁切为 `3:2` 或 `2:3`，目标比例素材会完整保留。
- 色板区域使用纯色，不叠加渐变。
- 五个候选色按照片中占比最高的色值生成，并尽量选择不同色系。
- 内容文本是一个支持换行的编辑框，默认填入自动提取的主色 HEX。

## 本地运行

```bash
npm run serve
```

默认地址：

```text
http://localhost:5173
```

如果要用 iPhone 访问，把 `localhost` 换成电脑在同一 Wi-Fi 下的局域网 IP。

## 验证

```bash
npm test
```

## 部署

推送到 `main` 分支后，GitHub Actions 会先运行测试，再将仓库根目录作为静态站点发布到 GitHub Pages。

## 保存

生成后在手机浏览器里长按成图，选择保存到相册。
